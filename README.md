# lnkCore

Generate aesthetic "core" posters from a keyword.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local`:

```bash
UNSPLASH_ACCESS_KEY=YOUR_KEY
```

3. Start the dev server:

```bash
npm run dev
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
